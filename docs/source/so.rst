==================================
Retrieve Remote Schema.org JSON-LD
==================================

The ``get-so`` subcommand retrieves remote JSON-LD.

*****
Usage
*****

::

    Usage: willitsync get-so [OPTIONS]

    Options:
      -u, --url TEXT  URL of landing page to try  [required]
      --help          Show this message and exit.
